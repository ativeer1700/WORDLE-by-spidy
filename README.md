# WORDLE:-
I made a web based clone of wordle game.

# Key deliverables:-
1. A intro landing page with my brief intro
2. A separate page to play the game
3. A clean, neat UI
4. User gets 6 tries to guess a 5 letter word
5. On every guess, the user is told whether:-
   .letter is correct
   .letter and position is correct
   .letter is incorrect
